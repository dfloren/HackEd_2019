Project Name: Profolio


Our entry for the Servus Credit Union Challenge in HackED 2019 Edmonton, a 24-hour hackathon.
This website intends to manage portfolios by analyzing the client's spending habits and
utlizing "N-assets mean-variance analysis" to determine optimal investments.

Sources:

www.w3schools.com

https://www.youtube.com/watch?v=UIJKdCIEXUQ

https://getbootstrap.com/docs/4.2/examples/starter-template

Running Instructions/ Notes:

The ff. must be installed:

1. python
2. pandas library for python
3. flask_wtf


To run:
Open command prompt, navigate to directory 'myflaskapp' containing the file 'app.py' and
type in the command 'python app.py'. 
Navigate to the local host using any web
browser with 'localhost:5000'
